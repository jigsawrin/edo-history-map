# プライバシーポリシー

「いま・むかし地図」(以下「本アプリ」)は、静的な Web アプリであり、
運営者側のサーバーやデータベースを持ちません。

## 運営者が収集しない情報

本アプリの運営者は、以下を含む利用者データを収集・保存しません。

- 位置情報(緯度・経度)
- 氏名、メールアドレスなどの個人情報
- ユーザー識別子、デバイス指紋
- 閲覧履歴、行動ログ

本アプリには、アクセス解析(Google Analytics 等)、広告タグ、
トラッカー、SNS 埋め込み、外部エラー送信(Sentry 等)、Cookie、
localStorage / sessionStorage / IndexedDB の使用は一切ありません。
ログイン機能、フォーム送信、コメント機能もありません。

## 位置情報の扱い

- ページを開いただけでは位置情報を要求しません。
- 「現在地を表示」ボタンを押し、説明に同意した場合のみ、
  ブラウザの位置情報 API(`getCurrentPosition`)を **1回** 呼び出します。
  継続的な追跡(`watchPosition`)は行いません。
- 取得した緯度・経度はブラウザのメモリ内でのみ使用し、
  独自サーバーへの送信・保存、URL への埋め込み、Cookie /
  localStorage への保存、コンソールへの出力は行いません。
- 現在地マーカーはページの再読み込みで消えます。
- 位置情報の許可はブラウザ・OS の設定でいつでも取り消せます。
  許可しなくても、地図閲覧機能はすべて利用できます。

## 外部への通信(誤解のないように)

「何も外部へ送信されない」わけではありません。以下の通信が発生します。

1. **地図タイル提供者(国土地理院)**
   地図表示のため、ブラウザから地理院タイルのサーバー
   (cyberjapandata.gsi.go.jp)へタイル画像の要求が送信されます。
   この要求には、IP アドレスと、表示中の地域に対応するタイル座標が
   含まれます。現在地を表示した場合、その付近のタイルが要求されます。
2. **ホスティング事業者(GitHub Pages / GitHub社)**
   本アプリの HTML・JavaScript・データの配信元である GitHub が、
   セキュリティ目的等でアクセス情報(IP アドレス等)を記録する場合が
   あります。詳細は GitHub のプライバシーステートメントをご確認ください。

上記以外の外部通信先はありません(Content Security Policy で制限しています)。

## 静的地点一覧

`/places/`以下の地点一覧は、承認済みデータからビルド時に作成した静的HTMLです。
JavaScript、検索フォーム、位置情報API、Cookie、localStorage / sessionStorage /
IndexedDB、アクセス解析、広告、外部画像、外部フォントを使用しません。閲覧中の
検索語や操作状態を保存・送信する機能もありません。

一覧ページ自体の配信ではGitHub Pagesへの通信が発生します。利用者が出典リンクを
選んだ場合に限り、明示された外部サイトへ移動し、そのサイトとの通信が発生します。

## 歴史年表

地図版の年表検索、時代区分・種別の絞り込み、選択中の年表項目は、URL、Cookie、
localStorage、sessionStorage、IndexedDBへ保存しません。年表の利用による追加API通信や
実行時fetchはありません。`/timeline/`以下の静的年表はJavaScript、フォーム、位置情報、
外部画像、外部フォントを使用せず、利用者が固定出典リンクを選んだ場合だけ外部サイトへ移動します。

## お問い合わせについて

GitHub リポジトリの Issue 等で利用者自身が送信した情報は、
GitHub のサービス上で公開・処理されます。本アプリの動作による
データ収集とは別の扱いになります。個人情報を Issue に書かないでください。

## 古地図ラスターパック

承認済み古地図が将来導入された場合も、定義・manifest・タイルは自サイトと同一originから
読み込み、古地図のための外部画像通信は行いません。選択シート、不透明度、表示モード、
読込状態はメモリ内だけで扱い、URL、Cookie、localStorage、sessionStorage、IndexedDBへ
保存しません。Service Workerや永続キャッシュも追加しません。現在は本番承認済み画像がなく、
古地図タイル通信自体が発生しません。候補台帳に外部所蔵館の資料URLを記録しても、
地図版の実行時通信先には追加しません。
