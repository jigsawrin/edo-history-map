# 滋賀・戦国 地点調査・採否監査

調査日: 2026-07-17

対象: 滋賀県の戦国期に関係する城郭、合戦、政治、寺社、居館、交通拠点

結論: high / medium の根拠を持つ36地点を採用し、実装ゲート（30件以上）を通過。

## 判断方法

- 歴史的根拠は滋賀県公式「近江の城郭」ガイド群、文化庁、長浜市の資料を優先した。
- 現在位置は公的所在地、現存遺構、考古遺跡、NII／CODH地名情報、国土地理院住所検索を照合した。
- 山城、戦場、寺域、城下町は面や範囲を一点へ縮約するため、代表位置としてmediumにした。
- highは、現在の寺社、館跡、現存遺構など、公的資料で現地点を直接確認できる場合に限定した。
- low、出典不明、現在位置を一点へ安全に定められない候補は公開しない。
- 説明文は事実を基に独自執筆し、参照先の原文・画像・HTML・PDFは転載していない。

## 不採用候補

| 候補 | 不採用理由 |
|---|---|
| 木村城跡 | 公式ガイド上の歴史的言及は確認できたが、現在位置を一点に固定する公的根拠を今回の調査範囲で確定できなかった。 |
| 常楽寺湊 | 湖岸変化を含む歴史的範囲で、現代座標の代表点を誤解なく示す根拠が不足した。 |
| 清水山城跡 | 広い山城域に対する現在位置資料の照合が採用基準へ達しなかった。 |
| 和田城跡 | 同名・関連地点の識別と現在位置の独立確認が不足した。 |
| 箕作城跡 | 歴史的根拠はあるが、採用座標の公的な位置根拠を固定できなかった。 |
| 田中城跡 | 名称と所在地の候補は得たが、史跡範囲と代表座標の関係を十分に説明できなかった。 |
| 今浜城跡 | 長浜城との関係を含む歴史的位置の扱いに解釈余地があり、別地点としての座標確度が不足した。 |
| 竹生島関連地点 | 戦国史との関係はあるが、今回の城郭・合戦・政治・寺社地点として一点を選ぶ根拠と説明範囲を確定しなかった。 |

不採用は史跡の重要性を否定する判断ではなく、本アプリの「公的・学術根拠と現在座標を地点単位で監査できる」という公開基準に届かなかったことを意味する。
